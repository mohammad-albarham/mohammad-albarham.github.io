Project's Details

This project is a MATLAB implementation of the Structure from Motion (SfM) algorithm. The SfM algorithm is a photogrammetric range imaging technique for estimating three-dimensional structures from two-dimensional image sequences that may be coupled with local motion signals. It is studied in the fields of computer vision and visual perception. The project is implemented using MATLAB and the code is available on both Github and Mathworks. The project was a final project for Computer Vision Course at Chalmers University of Technology.
